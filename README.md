# Datadog to Kibana

This repo aims to fetch dashboard and monitors created in Datadog and recreate them into Kibana

## Getting started

- clone this repository
- copy `.envrc.sample` to `.envrc`
- load it with direnv
- run: `yarn install`
- run: `yarn dev`
- you are done :clap:

## Build

- run: `yarn build`

## Run

- run: `yarn start`
